# 대시보드 디자인 자료 정리

이 폴더는 production 실행 경로가 아니라 dashboard prototype과 archive snapshot을 보관하는 공간이다.

## 현재 참조본

- `샘플2/project/Dirong Dashboard Explorations.html`
- `샘플2/project/Dirong Hi-Fi.html`
- `샘플2/project/dirong-components.css`
- `샘플2/project/dirong-screens.js`
- `샘플2/project/dirong-tokens.css`

## Archive Snapshot

- `샘플/`: 초기 hi-fi handoff snapshot. 현재 구현 기준으로 직접 사용하지 않는다.
- `와이어프레임/`: wireframe handoff snapshot. 현재 구현 기준으로 직접 사용하지 않는다.

새 dashboard 작업을 시작할 때는 먼저 현재 참조본을 확인하고, archive snapshot은 변경 이유를 추적할 때만 비교 대상으로 사용한다.
